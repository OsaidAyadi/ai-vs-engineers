# AI Can Write Code… But Can It Survive Production? 🤖🔥

A fun presentation about **how software engineers can work with AI and still stay important.**

Built as a **web presentation using HTML, CSS, and JavaScript.**

Because now AI can write code… but someone still has to fix the bugs 😅

---

# 🤔 The Question

Today AI can:

- write code
- explain code
- create tests
- answer developer questions

So people ask:

> **Will AI replace software engineers?**

Short answer: **No.**

But engineers who **use AI well** will work much faster.

---

# 💡 The Idea

AI is a tool.

Good engineers still need to:

- solve problems
- design systems
- review code
- make decisions
- fix production issues

AI can help, but **it cannot replace real engineering thinking.**
